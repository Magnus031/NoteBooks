    Introduction

    Redis 是一个后端开发常用的中间件，用于缓存数据库。它的全称其实是 (Remote Dictionary Server) 远程字典服务。

    为什么会引入呢？

    • 分布式缓存，提高数据访问速度，分担Mysql的压力
    • 因为我们平时的应用基本是都是读多写少，那么Mysql更侧重于存储，一致性，硬盘，而Redis更侧重于读取，所以可以提高效率

      - 硬件角度 : Redis -> memory ,Mysql -> disk
